# 安装

让我们在不到 1 分钟的时间内安装 pcl.js。

## NPM

```bash
# NPM
npm install pcl.js

# Yarn
yarn add pcl.js
```

## CDN

### 开发环境

```html
<script src="https://cdn.jsdelivr.net/npm/pcl.js/dist/pcl.js"><script>
```

### 生产环境

```html
<script src="https://cdn.jsdelivr.net/npm/pcl.js/dist/pcl.min.js"><script>
```
